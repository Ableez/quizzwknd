# Quiz Web App

This is a web app built with ReactJS and NodeJS that allows users to take a multiple-choice quiz on general knowledge. The app uses the Open Trivia API to            fetch  questions and presents them to the user one at a time. The user can select an answer for each question and see their score at the end of the quiz.
## Features

    1 Fetches questions from the Open Trivia API
    2 Presents questions one at a time with multiple-choice answers
    3 Keeps track of the user's score as they answer questions
    4 Allows users to enter username and email addresses
    5 Allows users to logout from the App
    6 Responsive design using Tailwind CSS

## Getting Started
To run this app on your local machine, you will need to have Node.js installed. You can follow these steps:

Clone the repository
```Terminal
git clone //
```
Install the necessary dependencies
```Terminal
npm install
``` 
Start the App
```Terminal
npm run dev
``` 
Open your browser and visit
```Terminal
http://localhost:5173
```
