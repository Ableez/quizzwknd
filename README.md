#Quiz Web App

This is a web app built with ReactJS and NodeJS that allows users to take a multiple-choice quiz on various topics. The app uses the Open Trivia API to fetch questions and presents them to the user one at a time. The user can select an answer for each question and see their score at the end of the quiz.
Features

    - Fetches questions from the Open Trivia API
    - Presents questions one at a time with multiple-choice answers
    - Keeps track of the user's score as they answer questions
    - Allows the user to restart the quiz at any time
    - Responsive design using Tailwind CSS

###How to run the app

    - Clone the repository: git clone https://github.com/your-username/quiz-web-app.git
    - Navigate to the project directory: cd quiz-web-app
    - Install the dependencies: npm install
    - Start the app: npm start
    - Open your browser and go to http://localhost:3000
