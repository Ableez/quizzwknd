#Quiz Web App

This is a web app built with ReactJS and NodeJS that allows users to take a multiple-choice quiz on general knowledge. The app uses the Open Trivia API to fetch questions and presents them to the user one at a time. The user can select an answer for each question and see their score at the end of the quiz.

###Features

    - Fetches questions from the Open Trivia API
    - Presents questions one at a time with multiple-choice answers
    - Keeps track of the user's score as they answer questions
    - Allows users to enter username and email addresses
    - Allows users to logout from the App
    - Responsive design using Tailwind CSS

###How to run the app

    - Clone the repository: git clone https://github.com/your-username/quiz-web-app.git
    - Navigate to the project directory: cd quiz-web-app
    - Install the dependencies: npm install
    - Start the app: npm start
    - Open your browser and go to http://localhost:3000

###Dependencies

    1 react
    2 react-dom
    3 react-scripts
    4 axios
    5 tailwindcss
    6 lodash

###Getting Started
To run this app on your local machine, you will need to have Node.js installed. You can follow these steps:

    1 Clone the repository
    2 Navigate to the project directory ```cd ./quizzle``` in your terminal
    3 Run ```npm install``` to install the necessary dependencies
    4 Run ```npm run dev``` to start the app
    5 Visit http://localhost:5173 in your browser to view the app
